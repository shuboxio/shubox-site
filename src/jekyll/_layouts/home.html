In the `div` above you'll see that there is a data attribute called `data-shubox-progress`. Shubox will update that data attribute with the upload percentage. This allows you to style your element how you'd like, based on the upload progress. It's how the "fill up" effect is accomplished in this particular demo. *Note:* You do NOT have to add that attribute for it to work - it will be added automatically upon file upload. Also of now `previewsContainer:false`. Setting `previewsContainer` to false will keep the element from inserting the preview template upon initial upload.
