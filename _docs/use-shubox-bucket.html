Set Up Your Shubox Bucket

Visit the bucket subscription page and choose what size/plan bucket you would like. Information on each plan is available on that page.

On your subscriptions page, there will be a link to create your first bucket. Visit the form, where you will be able to select:

  • ACL (private or public)
  • Region
  • A list of domains you would like to have access via the JS SDK
